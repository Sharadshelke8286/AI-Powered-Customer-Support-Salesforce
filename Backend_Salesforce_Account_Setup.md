## Salesforce Account Setup

## 📌 Introduction

Salesforce is a cloud-based Customer Relationship Management (CRM) platform that enables organizations to manage customer interactions, automate business processes, and leverage AI-powered tools to improve productivity.

In this project, Salesforce is used as the core platform to implement AI-Powered Customer Support & Service Automation using Prompt Builder, Flow Automation, and Lightning customization.

---

## 🔹 What Is Salesforce?

Salesforce is a customer success platform designed to help organizations:

- Sell products and services
- Provide customer support
- Automate service workflows
- Analyze performance through reports and dashboards
- Store data securely in the cloud

It provides standard and customizable features that allow businesses to manage customer relationships efficiently from anywhere.

---

## 🔹 Activity 1: Creating a Salesforce Developer Account

To begin development, a Salesforce Developer Organization (Org) was created.

### Steps Followed:

1. Visited the official Salesforce Developer Signup page:
   https://developer.salesforce.com/signup

2. Filled in the required details:
   - First Name & Last Name
   - Email Address
   - Role: Developer
   - Company: College Name
   - Country: India
   - Postal Code: PIN Code
   - Username: Format → username@organization.com  
     (Note: This does not need to be a real email address.)

3. Clicked on **Sign Me Up** to complete the registration process.

---

## 🔹 Activity 2: Account Activation

After registration:

1. Checked the email inbox used during signup.
2. Clicked on the **Verify Account** link received from Salesforce.
3. Created a secure password.
4. Set a security question and answer.
5. Clicked **Change Password**.
6. Successfully redirected to the Salesforce Setup Home Page.
